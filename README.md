
# DrawBoard
A Drawing app for android , built using kotlin and Canvas

## Features
- Draw in different Sizes.
- Undo & Redo.
- Choose different colours.
- Different Brush Sizes
- Share among Friends.
- Choose Different Background From Gallery.
